h2. cobagi

config backups using git(hub)

h3. Introduction

Backup der Einstellungen (Konfigurationsdateien) eines Linuxsystems mit Hilfe von github.com

h3. Create Github repository

"https://github.com/repositories/new (Create repository)":url

h3. Local setup

Download and install "http://git-scm.com/download (Git)":url

h4. Configure login

  $ git config --global user.email "your@example.com"
  $ git config --global user.name "Inte"

h4. Initialise repo

  $ su - 
  $ cd /
  $ git init
     
h4. Disable global tracking

  $ echo -e "bin/ \nboot/ \ndev/ \netc/ \n.git/ \nhome/ \nlib/ \nmedia/ \nmnt/ \nopt/ \nproc/ \nroot/ \nsbin/ \nsys/ \ntmp/ \nusr/ \nvar/" >> /.git/info/exclude

h4. Add config files

  $ git add /etc/conf.d/local.start
  $ git commit -m 'first commit'
  $ git remote add origin git@github.com:Inte/cobagi.git
  $ git push origin master

h4. When you're done
